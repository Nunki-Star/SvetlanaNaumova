Блок навигации

  • Ширина 15еm
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните
    контентом достаточным,
    чтобы по высоте он начинал
    захватывать вторую статью,
    как показано в данном примере

Должен присутствовать цвет фона Во всех "статьях" сделайте отступ снизу от соседей равным .2ет, при помощи свойства margin- bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутрь контентом
